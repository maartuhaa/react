# Fridge Recipe Finder

This is a small React project where you can find recipes based on ingredients you already have at home.

The idea is simple: instead of searching for recipes and then buying everything, you start with what you already have in your fridge.

---

## What you can do

- enter ingredients (for example: tomato, cheese)
- get a list of recipes
- open a recipe in a popup
- see:
  - which ingredients you already have
  - which ones are missing
  - cooking instructions
  - link to full recipe

---

## Tech used

- React  
- JavaScript  
- CSS  
- Spoonacular API
